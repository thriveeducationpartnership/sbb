Network Certificate Required

To access the Thrive network you must install the security certificate below.

Automatic Install (Windows)

If you are using a Windows device you can install the certificate automatically.

Install Certificate
Important Windows Notes:
1. After downloading, your browser may try to block or delete the file because it is unsigned. You will need to keep the file before running it.

2. Windows may display a SmartScreen warning. Click More Info → Run Anyway.

3. The certificate installs for your current user account. Administrator permissions are not required.

Manual Installation

If the installer does not work you can install the certificate manually.

Download Certificate

Instructions by Device

Windows

  1. Download the certificate
  2. Double-click the file
  3. Select Install Certificate
  4. Choose Current User
  5. Select Trusted Root Certification Authorities
  6. Finish the wizard

macOS

  1. Download the certificate
  2. Open the file
  3. Add to System Keychain
  4. Set trust to Always Trust

iPhone / iPad

  1. Download the certificate
  2. Open Settings
  3. Select Profile Downloaded
  4. Install the profile
  5. Enable trust in Certificate Settings

Android

  1. Download the certificate
  2. Open Security Settings
  3. Select Install Certificate
  4. Choose the downloaded file

Why is this required?

The Thrive network uses secure web filtering to support safeguarding requirements. Installing the certificate allows encrypted websites (HTTPS) to work correctly while connected to the network.
