Network Certificate Required

To access the internet on the Thrive network you must install the security certificate below.

Automatic Install (Windows)

If you are using Windows you can install the certificate automatically.

Download Installer
Note: Windows may display a SmartScreen warning. Click More Info → Run Anyway.

The certificate will be installed for your current user account. Administrator rights are not required.

Manual Installation

If you prefer, download the certificate and install it manually.

Download Certificate

Instructions by Device

Windows

  1. Download the certificate
  2. Double-click the file
  3. Select Install Certificate
  4. Choose Current User
  5. Select Trusted Root Certification Authorities
  6. Finish

macOS

  1. Download the certificate
  2. Open the file
  3. Add to System Keychain
  4. Set trust to Always Trust

iPhone / iPad

  1. Download the certificate
  2. Open Settings → Profile Downloaded
  3. Install profile
  4. Enable trust in Certificate Settings

Android

  1. Download certificate
  2. Open Security Settings
  3. Select Install Certificate
  4. Choose the downloaded file

Why is this required?

The Thrive network uses secure web filtering to meet safeguarding requirements. Installing the certificate allows secure websites (HTTPS) to function correctly while connected to the network.
