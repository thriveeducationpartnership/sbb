Network Certificate Required

To access the internet on this network you must install the security certificate.

Install Certificate

Download Certificate

Windows Instructions

  1. Download the certificate
  2. Open the downloaded file
  3. Select Install Certificate
  4. Choose Current User
  5. Select Place in Trusted Root Certification Authorities
  6. Finish the wizard
Administrator rights are not required. The certificate installs only for your user account.

macOS Instructions

  1. Download the certificate
  2. Open the file
  3. Add to System Keychain
  4. Open Keychain Access
  5. Set the certificate to Always Trust

iPhone / iPad Instructions

  1. Download the certificate
  2. Open Settings
  3. Select Profile Downloaded
  4. Install the profile
  5. Go to General → About → Certificate Trust Settings
  6. Enable trust for the certificate

Android Instructions

  1. Download the certificate
  2. Open Security Settings
  3. Select Install Certificate
  4. Choose the downloaded file

Why is this required?

This network uses secure filtering to meet safeguarding requirements. Installing the certificate allows secure websites (HTTPS) to work correctly while connected.

If you have issues installing the certificate please contact IT support.
