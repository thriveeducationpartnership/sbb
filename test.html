Network Certificate Required

To access the internet on this network you must install the security certificate.

Automatic Install

If you are using Windows you can install the certificate automatically.

Install Automatically (Windows)
Windows Security Warning? If Windows shows a SmartScreen warning: 1. Click More Info 2. Click Run Anyway

Manual Installation

If the automatic installer does not work, download the certificate and install it manually.

Download Certificate

Installation Instructions

Windows

  1. Download the certificate
  2. Double-click the file
  3. Select Install Certificate
  4. Choose Local Machine
  5. Install into Trusted Root Certification Authorities

macOS

  1. Download the certificate
  2. Open the file
  3. Add to System Keychain
  4. Set trust to Always Trust

iPhone / iPad

  1. Download the certificate
  2. Open Settings
  3. Tap Profile Downloaded
  4. Install the profile
  5. Enable trust in Certificate Settings

Android

  1. Download the certificate
  2. Open Security Settings
  3. Select Install Certificate
  4. Choose the downloaded file

Why is this required?

This network uses secure filtering to protect users and comply with safeguarding requirements. Installing the certificate allows encrypted websites (HTTPS) to work correctly while connected to the network.

If you have any issues installing the certificate please contact IT support.
