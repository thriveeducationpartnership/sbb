Network Certificate Installation

This page allows you to install the required network certificate for Thrive Education Partnership.

Automatic Install (Windows)

If you are using a Windows device you can install the certificate automatically.

Install Certificate
Important for Windows users:
1. After downloading, your browser may try to block or delete the installer because it is unsigned — keep the file before running it.

2. Windows may display a SmartScreen warning — click More Info → Run Anyway.

3. The certificate installs for your current user account. Administrator permissions are not required.

Manual Installation

If the automatic installer does not work you can install the certificate manually.

Download Certificate

Installation Instructions by Device

Windows

  1. Download the certificate
  2. Open the downloaded file
  3. Select Install Certificate
  4. Choose Current User
  5. Select Trusted Root Certification Authorities
  6. Finish the wizard
You may need to restart or reload your browser for HTTPS sites to work.

macOS

  1. Download the certificate
  2. Open the file
  3. Add to System Keychain
  4. Set trust to Always Trust
You may need to restart or reload your browser for HTTPS sites to work.

iPhone / iPad

  1. Download the certificate
  2. Open Settings
  3. Select Profile Downloaded
  4. Install the profile
  5. Enable trust in Certificate Settings
You may need to restart or reload your browser for HTTPS sites to work.

Android

  1. Download the certificate
  2. Open Security Settings
  3. Select Install Certificate
  4. Choose the downloaded file
You may need to restart or reload your browser for HTTPS sites to work.

Why is this required?

Thrive Education Partnership is committed to safe and filtered internet access for all learners and staff. Installing this certificate allows secure websites (HTTPS) to work correctly on the network.
